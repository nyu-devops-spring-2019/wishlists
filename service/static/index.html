Wishlist Demo REST API Service

GET /wishlists to see all the wishlists.

GET /wishlists?name=Shoes retrieves a wishlist from the DB using name. Shoes in this case

GET /wishlists?customer_id=3 retrieves a wishlist from the DB using customer_id

GET /wishlists/id retrieves a wishlist from the DB using db id

PUT /wishlists/{id} updates a wishlists using an ID.

DELETE /wishlists/{id} removes a wishlist from the DB using an ID.

POST /wishlists creates a new wishlist.

Example Data:

    {
        "name": "Shoes",
        "customer_id": "3"
    }
